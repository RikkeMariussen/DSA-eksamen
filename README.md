# DSA-eksamen
Eksamensprojekt for valgfaget Datastrukturer og algoritmer, Datamatiker uddannelsen på Erhvervsakademiet København, Lyngby Campus.

# Hvad skal readme indeholde: **HUSK AT SLETTE**
- Repositoriet skal indeholde en README.md med samme information som forsiden af rapporten:
    - Et screenshot af den kørende applikation
    - En kort beskrivelse for “almindelige mennesker”
    - Et link til den deployede udgave.
    - README.md må også indeholde informationer om de anvendte algoritmer og datastrukturer, og beskrivelser af hvordan man kan køre visualiseringen lokalt.
